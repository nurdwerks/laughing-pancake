Download in PGN format

At this section, the database is divided into different time ranges in order to reduce the file size.

Monthly updates for the database can be found in a separate tab. They contain the games of the respective month. It is possible that these files will be uploaded again over time. The version number will then be adjusted and this will be indicated in the version history. The new process will mainly be necessary due to new games from the Lichess Broadcast system.

To unzip the files, Windows 7-Zip or another program that can handle the 7z format is required. On Mac, unpacking is possible with Keka, for example.

Caution

Using non-Latin characters (e.g. Arabic) in file paths can cause problems when unpacking archives under Windows 11. Please only use Latin characters in file paths and folder names to extract the archives.
